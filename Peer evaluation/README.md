# Subject 6: Around Simpson's Paradox

## Repository structure
- `data/`: dataset used in the analysis
- `scripts/`: R scripts for data processing and analysis
- `report/`: analysis and visualizations

## How to run
1. Clone this repository
2. Open the project in RStudio
3. Install the required packages
4. Run the analysis scripts in the following order:
   - `scripts/01_load_clean.R`
   - `scripts/02_mortality_tables.R`
   - `scripts/03_age_groups_simpson.R`
   - `scripts/04_logistic_regression.R`
